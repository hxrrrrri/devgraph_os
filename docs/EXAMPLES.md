# Examples

## Build And Explain

```bash
devgraph init
devgraph build
devgraph explain src/main.py
```

## Review

```bash
devgraph update
devgraph review --staged
```

Outputs:

- `.devgraph/reports/review.md`
- `.devgraph/reports/review.json`

## Dashboard

```bash
devgraph dashboard
```

Screenshot placeholders:

- Project Overview
- Graph View
- Review Lens
- Debug Lens
- Onboarding Lens
- Knowledge Lens
